# 작업 로그

## Usage
- 짧은 진행 로그, 관찰, 임시 blocker만 남긴다.
- durable한 사실이나 정책은 `docs/knowledge/` 또는 더 구체적인 canonical 문서로 승격한다.
- 새 작업을 시작할 때 오래된 항목은 정리하고 현재 작업에 필요한 로그만 유지한다.
