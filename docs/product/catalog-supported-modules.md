# Catalog Supported Modules

## 목적
운영에서 지원하는 도메인 모듈 목록과 사용자 관점의 차이를 기록한다.

## 표 템플릿

| slug | 노출 이름 | 주요 데이터 원천 | 비고 |
| --- | --- | --- | --- |
| `{{MODULE_SLUG}}` | `{{MODULE_LABEL}}` | `{{MODULE_SOURCE}}` | `{{MODULE_NOTE}}` |

## 규칙
- test-only 또는 fallback 모듈은 여기에 넣지 않는다.
- 지원 목록이 바뀌면 런타임 등록 위치와 함께 갱신한다.
