# Design Docs

## Core Rules
- `design-docs`는 작업 로그가 아니라 장기 재사용 가치가 있는 설계 결정을 저장한다.
- 파일명은 번호가 붙은 ADR 스타일을 사용한다.
- 각 문서는 problem, decision, impact를 분명히 적는다.
