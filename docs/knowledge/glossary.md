# Glossary

## canonical source
특정 사실이나 규칙의 단일 진실 공급원 문서.

## knowledge
여러 세션에 걸쳐 재사용할 장기 참조 지식 계층. 사용자 노출 정책, 장기 결정, 용어, 반복 패턴을 담는다.

## runtime context
현재 작업의 목표, 가정, blocker, 다음 행동을 담는 live 상태 문서. 기본 위치는 `docs/memory/runtime/context.md`다.

## daily log
짧은 진행 로그와 관찰 메모를 남기는 보조 runtime 문서. 기본 위치는 `docs/memory/runtime/dailylog.md`다.

## summary layer
canonical source를 대체하지 않고 위치와 읽기 순서만 안내하는 얇은 문서.
