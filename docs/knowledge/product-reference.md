# Product Reference

## 현재 확인한 사용자 표면
- 현재 저장소는 실행 서비스가 아니라 범용 문서 하네스 템플릿 저장소다.
- 사용자에게 직접 노출되는 주요 자산은 루트 `README.md`와 `Prompt/` 디렉터리의 Markdown 문서다.
- 확인된 핵심 입력 문서는 `Prompt/init.md`다.

## 현재 확인한 지원 범위
- 문서 하네스 초기화 프롬프트와 관련 설명 문서를 제공한다.
- 실행 가능한 API, 웹 UI, CLI 지원 범위는 확인되지 않았다.
