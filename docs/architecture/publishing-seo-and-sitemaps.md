# Publishing SEO And Sitemaps

## 목적
사용자 표면, SEO 메타, robots, sitemap 같은 생성 산출물의 구조를 설명한다.

## 핵심 규칙
- SEO 정책은 product와 architecture를 함께 읽고 판단한다.
- 빌드 산출물은 source-first로 다룬다.
- indexability, canonical, generated output 경로는 한 문서 세트에서 일관되게 관리한다.

## 이 프로젝트에서 채워 넣을 항목
- SEO 메타 조립 경로: `{{SEO_PATHS}}`
- generated output 경로: `{{GENERATED_OUTPUT_PATHS}}`
- 관련 라우트 / 명령 진입점: `{{PUBLISHING_ENTRYPOINTS}}`
